# dexcli-core

**Terminal agent team manager — turns CLI agents into coordinated teams.**

| | |
|---|---|
| Layer | L3 — Tool under DexSpine (L2) |
| Organisation | [Dex-RoboKnix](https://github.com/Dex-RoboKnix) |
| Edition | Core — Open Source |
| Status | Pre-release / In Development |

## Overview

DexCLI turns terminal-based LLM agents into coordinated teams. It manages agent pools, routes tasks, preserves sessions, parses output artefacts, and optionally accepts DexSpine `dex()` packets for deterministic task handoff. Gemini CLI is the first provider adapter; the architecture is provider-agnostic.

## OpenCore

This is the Core (open source) edition. An Enterprise edition with additional connectors, managed hosting, and commercial support is maintained separately under the same organisation.
